## Installation Steps

### On Android Phone

1. **Download the APK**
   - Click the download link above
   - Or visit: https://github.com/kathirvel-p22/FieldFresh
   - Click on `build/app/outputs/flutter-apk/app-release.apk`
   - Click "Download" button

2. **Enable Unknown Sources**
   - Go to Settings → Security
   - Enable "Install from Unknown Sources"
   - Or allow installation from Chrome/Files app

3. **Install**
   - Open the downloaded APK file
   - Tap "Install"
   - Wait for installation to complete
   - Tap "Open"

4. **Start Using**
   - Login with test accounts (see below)
   - Explore all features
   - Test the marketplace

## Troubleshooting

### Can't Download
- Check internet connection
- Try different browser
- Use mobile data instead of WiFi

### Can't Install
- Enable "Install from Unknown Sources"
- Check if you have enough storage (need ~150 MB)
- Try restarting phone

### App Won't Open
- Check Android version (need 5.0+)
- Clear cache and try again
- Reinstall the APK
